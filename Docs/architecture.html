🤖 Job Finder Agent — Phase 2 Design ▶ IN DESIGN

The agent interviews the user by voice and text, then autonomously calls this system's own APIs as GPT function-calling tools. It discovers institutions and jobs mid-conversation, builds resumes, drafts emails, and presents a review queue. The user approves and submits. See agent_vision.html for full spec.

⚡ API Endpoints — Phase 1 (all live)

GET    /api/domains/                    List all domains
POST   /api/domains/                    Create domain
PUT    /api/domains/{id}/activate       Set active domain
DELETE /api/domains/{id}                Delete domain

GET    /api/jobs/                       List jobs (active domain filtered)
GET    /api/jobs/new                    New jobs only
GET    /api/jobs/{id}                   Job detail
POST   /api/jobs/scan                   Trigger background scan
PUT    /api/jobs/{id}/status            Update job status

GET    /api/institutions/               List institutions (active domain)
POST   /api/institutions/               Add institution
PUT    /api/institutions/{id}/favorite  Toggle favorite
DELETE /api/institutions/{id}           Delete

POST   /api/resumes/build               Build resume via GPT-4o-mini ✓
GET    /api/resumes/                    List all resumes
GET    /api/resumes/{id}                Get resume
GET    /api/resumes/{id}/pdf            Download PDF
GET    /api/resumes/{id}/docx           Download DOCX

GET    /api/applications/               List applications
POST   /api/applications/               Create application
PUT    /api/applications/{id}           Update status/notes

GET    /api/profile/                    Get user profile
PUT    /api/profile/                    Update profile
GET    /api/profile/skills              List skills
POST   /api/profile/skills              Add skill
DELETE /api/profile/skills/{id}         Delete skill
GET    /api/profile/experience          List experience
POST   /api/profile/experience          Add experience
