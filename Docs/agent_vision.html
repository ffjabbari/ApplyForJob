🤖 Agent Vision

JobHunter AI — The Complete Agentic Workflow

🎯 The Vision in One Paragraph

A Job Finder Agent interviews you by voice and text. As you talk, it autonomously calls this system's own APIs to build itself — adding institutions, queuing jobs, building tailored resumes and cover letters, and drafting submission emails with your documents attached. Before anything is sent, it presents each item for your review. You approve, edit, or ask follow-up questions. When you press Submit, the agent sends your application — one at a time, reviewed and approved by you. You never manually fill a form. The conversation IS the workflow.

🎙️ Voice + Text Interface

The agent accepts both voice (Web Speech API / Whisper) and typed input. Responses are spoken back via text-to-speech and displayed in the chat.

🎙️ Voice Input (Whisper / Web Speech) 🔊 Voice Output (TTS) POST /api/agent/transcribe POST /api/agent/speak

🔧 Agent Tool Belt — APIs the Agent Calls Autonomously

The agent is given these tools as GPT function-calling definitions. It decides when and how to call them based on the conversation.
