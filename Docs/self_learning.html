🧠 Self-Learning Phase

JobHunter AI — AI Interview Agent that builds the system through conversation

📍 Phase Status

✓ Phase 1 — Core System ▶ Phase 2 — Self-Learning Agent ◷ Phase 3 — Real Scrapers ◷ Phase 4 — Auto Apply

Phase 1 is complete and running locally. The backend (FastAPI + SQLite), frontend (React + Vite), resume builder (GPT-4o-mini), and domain system are all live. Phase 2 introduces the AI Interview Agent — a conversational interface that replaces manual data entry by interviewing the user, discovering jobs and institutions in real time, and self-populating the system from the conversation.

💡 Core Concept — Conversation-Driven System Building

Instead of the user manually filling forms, the AI Agent interviews the prospect using job-specific templates. As the user answers, the agent simultaneously searches the web, discovers matching institutions and jobs, and proactively suggests adding them to the system. By the end of the session, the resume is built, institutions are populated, and jobs are queued — all from a single conversation.

🤖
Welcome! I'm going to help you find the perfect teaching position. Let's start with your background. What subjects are you most passionate about teaching?
👤
I love teaching software engineering, AI, and cloud architecture. I've been doing it informally for years at work.
🤖
Great answer! Software engineering + AI is a hot combination right now.
🔍 I just found 3 things based on your answer:
• Washington University in St. Louis is hiring an Adjunct in CS/AI
• UMSL has an open "Cloud Computing" lecturer position
• SLU's School of Engineering lists AI as a priority hire for Fall 2026

Want me to add these institutions to your list and queue those jobs?
👤
Yes, add all three!
🤖
✅ Added WashU, UMSL, and SLU to your institutions. 3 jobs queued.

Next question: What's your highest degree and field?

⚡ New API Endpoints — Phase 2

# Interview Sessions
POST   /api/interview/start          # Start session with template_id
POST   /api/interview/{id}/message   # Send user message, get AI response + discoveries
GET    /api/interview/{id}           # Get full session history
GET    /api/interview/{id}/summary   # Get extracted profile + queued jobs

# Resume Templates
GET    /api/templates                # List all templates
GET    /api/templates/{id}           # Get template with questions
POST   /api/templates                # Create custom template

# Discovery (called internally by agent)
POST   /api/discover/institutions    # Search web for institutions matching query
POST   /api/discover/jobs            # Search web for jobs matching profile
